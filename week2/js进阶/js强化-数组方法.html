JavaScript 数组方法大全

打开控制台（F12）查看完整运行结果 | 页面上为精选示例

■ 红色标签 = 改变原数组 ■ 绿色标签 = 不改变原数组 ■ 灰色标签 = 静态方法

1. forEach 不改变原数组

遍历每个元素，无返回值（返回 undefined），不能中途 break

const arr = [10, 20, 30]
arr.forEach((item, index) => {
  console.log(`索引${index}: ${item}`)
})
→ 索引0: 10 / 索引1: 20 / 索引2: 30
// 与 for...of 对比：for...of 可用 break
for (const item of arr) { if (item > 15) break }

2. map 不改变原数组

遍历并返回新数组，每个元素经回调处理，数组长度不变

const nums = [1, 2, 3]
nums.map(n => n * 2)
→ [2, 4, 6]
// 实战：提取对象数组中的某个字段
const users = [{name:'张三', age:20}, {name:'李四', age:25}]
users.map(u => u.name)
→ ["张三", "李四"]

3. filter 不改变原数组

筛选符合条件的元素，返回新数组，长度 ≤ 原数组

const nums = [1, 2, 3, 4, 5]
nums.filter(n => n > 3)
→ [4, 5]
// 实战：去除 falsy 值（最常用）
[0, 'hello', null, '', 42, undefined].filter(Boolean)
→ ["hello", 42]

4. find / findIndex 不改变原数组

find 返回第一个符合条件的元素，findIndex 返回其索引，找不到返回 undefined / -1

const users = [{id:1,name:'A'},{id:2,name:'B'},{id:3,name:'A'}]
users.find(u => u.name === 'A')
→ { id: 1, name: "A" } (只返回第一个)
users.findIndex(u => u.name === 'A')
→ 0
users.find(u => u.id === 999)
→ undefined (找不到)

5. some / every 不改变原数组

some "有一项满足即 true"，every "全部满足才 true"

const nums = [1, 2, 3, 4, 5]
nums.some(n => n > 3)
→ true (有大于3的)
nums.every(n => n > 0)
→ true (全部都大于0)
nums.every(n => n > 3)
→ false (不是全都大于3)
// 实战：验证数组中是否有空值
['a', 'b', ''].some(item => !item)
→ true (有空字符串)

6. reduce 不改变原数组

万能方法：累加、去重、分组、扁平化都能做。
reduce((累计值, 当前值) => ..., 初始值)

// 求和（最常见）
[1, 2, 3, 4].reduce((sum, n) => sum + n, 0)
→ 10
// 数组去重（面试常考）
[1,2,2,3].reduce((acc, n) => {
  if (!acc.includes(n)) acc.push(n)
  return acc
}, [])
→ [1, 2, 3]
// 分组：按年龄分组
[{n:'A',a:20},{n:'B',a:20},{n:'C',a:30}].reduce((acc, p) => {
  (acc[p.a] ??= []).push(p)
  return acc
}, {})
→ {20: [{n:"A",a:20},{n:"B",a:20}], 30: [{n:"C",a:30}]}

7. push / pop 改变原数组

push 尾部添加(返回新长度)，pop 尾部删除(返回被删元素)。栈结构

const arr = [1, 2]
arr.push(3, 4)
→ 返回 4, arr 变为 [1, 2, 3, 4]
const arr2 = [1, 2, 3]
const last = arr2.pop()
→ last = 3, arr2 变为 [1, 2]

8. unshift / shift 改变原数组

unshift 头部添加(返回新长度)，shift 头部删除(返回被删元素)。队列结构

const arr = [2, 3]
arr.unshift(0, 1)
→ 返回 4, arr 变为 [0, 1, 2, 3]
const arr2 = [1, 2, 3]
const first = arr2.shift()
→ first = 1, arr2 变为 [2, 3]
// 性能注意：unshift/shift 比 push/pop 慢
// 因为头部操作需要移动所有元素索引

9. splice(start, deleteCount, ...items) 改变原数组

万能增删改：删除/插入/替换，返回被删元素的数组

const arr = ['a', 'b', 'c', 'd']
// 删除：从索引1开始删2个
arr.splice(1, 2)
→ 返回 ["b", "c"], arr 变为 ["a", "d"]
const arr2 = ['a', 'd']
// 插入：deleteCount=0，在索引1处插入
arr2.splice(1, 0, 'b', 'c')
→ 返回 [], arr2 变为 ["a", "b", "c", "d"]
const arr3 = ['a', 'x', 'c']
// 替换：删1个，插入1个
arr3.splice(1, 1, 'b')
→ 返回 ["x"], arr3 变为 ["a", "b", "c"]

10. slice(start, end) 不改变原数组

截取子数组（浅拷贝），支持负数。左闭右开 [start, end)

const arr = ['a', 'b', 'c', 'd', 'e']
arr.slice(1, 3)
→ ["b", "c"]
arr.slice(-2)
→ ["d", "e"] (取最后两个)
// 浅拷贝数组（常用）
const copy = arr.slice()
→ copy 是 arr 的浅拷贝 [a,b,c,d,e]

11. sort(compareFn) 改变原数组

默认按字符串排序！数字排序需传比较函数。a-b 升序 / b-a 降序

// 陷阱：默认按字符串排！
[1, 30, 4, 21].sort()
→ [1, 21, 30, 4] 😱（按字符串"1","21","30","4"）
// 正确：传比较函数
[1, 30, 4, 21].sort((a, b) => a - b)
→ [1, 4, 21, 30] (升序)
[1, 30, 4, 21].sort((a, b) => b - a)
→ [30, 21, 4, 1] (降序)

12. reverse 改变原数组

反转数组顺序

const arr = [1, 2, 3]
arr.reverse()
→ arr 变为 [3, 2, 1]
// 不想改变原数组：先拷贝再反转
const reversed = [...arr].reverse()
→ 原数组不变，reversed 是反转后的新数组

13. includes / indexOf 不改变原数组

includes 判断是否存在(NaN能识别)，indexOf 返回位置(不能识别NaN)

[1, 2, 3].includes(2)
→ true
// includes 能正确判断 NaN
[NaN].includes(NaN)
→ true (indexOf 做不到！)
[NaN].indexOf(NaN)
→ -1 (NaN !== NaN，所以找不到)

14. flat(depth) / flatMap 不改变原数组

flat 按深度扁平化，flatMap = map + flat(1)

[1, [2, 3], [4, [5]]].flat()
→ [1, 2, 3, 4, [5]] (默认只摊1层)
[1, [2, [3]]].flat(2)
→ [1, 2, 3] (摊2层)
// 完全摊平：Infinity
[1,[2,[3,[4]]]].flat(Infinity)
→ [1, 2, 3, 4]
// flatMap：先 map 再 flat(1)
['hello', 'world'].flatMap(w => w.split(''))
→ ["h","e","l","l","o","w","o","r","l","d"]

15. join(分隔符) 不改变原数组

数组转字符串，split 的反向操作

['a', 'b', 'c'].join(',')
→ "a,b,c"
['h', 'e', 'l', 'l', 'o'].join('')
→ "hello"

16. concat 不改变原数组

合并数组，但展开运算符 [...a, ...b] 更常用

[1].concat([2, 3], [4])
→ [1, 2, 3, 4]
// 更推荐：展开运算符
[...[1], ...[2, 3], ...[4]]
→ [1, 2, 3, 4]

17. Array.from / Array.isArray 静态方法

Array.from 将类数组/可迭代对象转为数组，Array.isArray 判断类型

// 类数组 → 真数组
Array.from('hello')
→ ["h","e","l","l","o"]
// 生成序列 (第二个参数是 map)
Array.from({ length: 5 }, (_, i) => i + 1)
→ [1, 2, 3, 4, 5]
Array.isArray([1, 2])
→ true
Array.isArray({ length: 2 })
→ false (类数组不是真数组)

18. fill(value, start, end) 改变原数组

用指定值填充数组，左闭右开

new Array(5).fill(0)
→ [0, 0, 0, 0, 0]
['a', 'b', 'c', 'd'].fill('X', 1, 3)
→ ["a", "X", "X", "d"]

19. 链式调用 — 数据处理流水线

map/filter/reduce/sort 返回新数组，可以链式调用，像流水线一样处理数据

// 找出年龄≥18的用户名字，按年龄升序排列
const users = [
  { name: '张三', age: 16 },
  { name: '李四', age: 22 },
  { name: '王五', age: 18 },
  { name: '赵六', age: 30 },
]
const result = users
  .filter(u => u.age >= 18)
  .sort((a, b) => a.age - b.age)
  .map(u => u.name)
→ ["王五", "李四", "赵六"]
// 一行代码：求所有偶数的平方和
[1,2,3,4,5].filter(n=>n%2===0).map(n=>n*n).reduce((s,n)=>s+n,0)
→ 20 (2² + 4² = 4 + 16 = 20)

20. 高频实战场景

// 数组去重（3种方式）
const arr = [1, 2, 2, 3, 3, 1]
// 方式1：Set（最简洁）
[...new Set(arr)]
→ [1, 2, 3]
// 方式2：filter + indexOf
arr.filter((item, index) => arr.indexOf(item) === index)
→ [1, 2, 3]
// 对象数组按属性去重（按id）
[{id:1},{id:2},{id:1}].filter((item,i,self)=> self.findIndex(t=>t.id===item.id)===i)

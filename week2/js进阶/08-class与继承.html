08 — class 语法糖与继承

class 本质上就是构造函数 + 原型链的语法糖。下面把 class 写法和等价的构造函数写法对比来看。

一、class 基本语法 vs 构造函数

class 写法

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  // 方法自动挂到 Person.prototype 上
  sayHi() {
    return this.name + ' 你好，' + this.age + ' 岁';
  }
  // static = 类自身的属性/方法，不在实例上
  static species = '人类';
}

等价的构造函数写法

function Person(name, age) {
  this.name = name;
  this.age = age;
}
Person.prototype.sayHi = function() {
  return this.name + ' 你好，' + this.age + ' 岁';
};
Person.species = '人类';

二、extends 继承 vs 原型继承

class extends 写法（现代写法）

class Animal {
  constructor(name) { this.name = name; }
  eat() { return this.name + ' 在吃东西'; }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name); // ← 必须调用 super() 且在使用 this 之前
    this.breed = breed;
  }
  bark() { return this.name + ' 汪汪叫'; }
  // 可以覆盖父类方法
  eat() { return super.eat() + '（吃狗粮）'; }
}

const d = new Dog('旺财', '金毛');
d.eat();  // "旺财 在吃东西（吃狗粮）"
d.bark(); // "旺财 汪汪叫"
d instanceof Dog;    // true
d instanceof Animal; // true

三、getter / setter — 计算属性

用 get/set 创建像属性一样调用的方法

class Temperature {
  constructor(celsius) {
    this._celsius = celsius; // _ 开头约定为"私有"
  }
  get celsius() { return this._celsius; }
  set celsius(val) {
    if (val < -273.15) throw new Error('低于绝对零度！');
    this._celsius = val;
  }
  get fahrenheit() {
    return this._celsius * 9 / 5 + 32; // 计算属性，不用存
  }
}

const t = new Temperature(25);
console.log(t.fahrenheit); // 77 — 像属性一样访问，实际在计算方法
t.celsius = 100;           // 像赋值一样，实际在调用 setter

四、私有字段（#）— 真正的私有

用 # 声明的字段，外部完全无法访问

class BankAccount {
  #balance = 0; // 私有字段，外部 bank.#balance 会报语法错误

  constructor(initial) { this.#balance = initial; }

  deposit(n) { this.#balance += n; }
  getBalance() { return this.#balance; }
}

const bank = new BankAccount(1000);
bank.deposit(500);
console.log(bank.getBalance()); // 1500
console.log(bank.#balance);     // ❌ SyntaxError — 外部访问不到

五、class 方法的 this 绑定问题

class 方法内部的 this 也会丢失（与普通函数一样）

class Counter {
  constructor() { this.count = 0; }
  increment() { this.count++; }
}
const c = new Counter();

// ❌ 方法被赋值后 this 丢失
const fn = c.increment;
fn(); // TypeError: Cannot read properties of undefined

// ✅ 解决方案1：bind
const boundFn = c.increment.bind(c);
// ✅ 解决方案2：箭头函数在 class field 中
class CounterFixed {
  count = 0;
  increment = () => { this.count++; }; // 自动绑定 this
}

JavaScript 字符串方法大全

打开控制台（F12）查看所有运行结果 | 页面上为精选示例

1. charAt(index) 获取字符

返回指定索引位置的字符，越界返回空字符串

const str = 'Hello 世界'
str.charAt(0)
→ "H"
str.charAt(100)
→ "" (越界返回空字符串，不报错)
// 用索引访问也行，但越界返回 undefined
str[0]   // "H"
str[100] // undefined

2. charCodeAt(index) 获取编码

返回 UTF-16 码元（0-65535），越界返回 NaN

'ABC'.charCodeAt(0)
→ 65 (A 的 ASCII 码)
'ABC'.charCodeAt(1)
→ 66 (B 的 ASCII 码)
// codePointAt 能正确处理 emoji（>65535）
'𠮷'.codePointAt(0)
→ 134071 (大于 65535 的码点)

3. indexOf / lastIndexOf 查找位置

返回子串首次/末次出现的位置，找不到返回 -1

'hello world hello'.indexOf('hello')
→ 0 (首次出现位置)
'hello world hello'.indexOf('hello', 2)
→ 12 (从索引2开始找，跳过第1个)
'hello world hello'.lastIndexOf('hello')
→ 12 (从末尾开始找)
'hello'.indexOf('xxx')
→ -1 (找不到)

4. includes / startsWith / endsWith 布尔判断

ES6 新增，返回布尔值，比 indexOf 更语义化

'hello world'.includes('world')
→ true
'hello world'.startsWith('hel')
→ true
'hello world'.endsWith('rld')
→ true
// 第二个参数：从哪个位置开始算
'hello'.startsWith('ll', 2)
→ true (从索引2开始是"llo", 以"ll"开头)

5. slice(start, end) 截取

最常用！支持负数（从末尾数），左闭右开 [start, end)

'JavaScript'.slice(0, 4)
→ "Java"
'JavaScript'.slice(4)
→ "Script" (省略 end 截到末尾)
'JavaScript'.slice(-6)
→ "Script" (负数从末尾倒数)
'JavaScript'.slice(-6, -1)
→ "Scrip" (倒数第6位到倒数第1位)

6. substring(start, end) 截取

类似 slice，但不支持负数，且会自动交换大小参数

'JavaScript'.substring(0, 4)
→ "Java"
// 与 slice 的关键区别：负数当 0 处理
'JavaScript'.substring(-5)
→ "JavaScript" (负数→0，即 substring(0))
// 自动交换：end < start 时会调换
'JavaScript'.substring(4, 0)
→ "Java" (等价于 substring(0,4))

7. toUpperCase / toLowerCase 大小写

全转大写/全转小写，常用于搜索时忽略大小写

'Hello World'.toUpperCase()
→ "HELLO WORLD"
'Hello World'.toLowerCase()
→ "hello world"
// 实战：忽略大小写搜索
'JavaScript'.toLowerCase().includes('SCRIPT'.toLowerCase())
→ true

8. trim / trimStart / trimEnd 去空白

去除空白字符（空格、制表符、换行符等），常用于表单输入处理

' hello '.trim()
→ "hello"
' hello '.trimStart()
→ "hello " (只去左边)
' hello '.trimEnd()
→ " hello" (只去右边)
// 表单验证常用
const input = ' '
input.trim().length === 0
→ true (判断是否为空输入)

9. replace / replaceAll 替换

replace 只替换第一个，replaceAll 替换全部（ES2021）

'banana'.replace('a', 'X')
→ "bXnana" (只替换第一个 a)
'banana'.replaceAll('a', 'X')
→ "bXnXnX" (全部替换)
// replace 配合正则 /g 也能全替换
'banana'.replace(/a/g, 'X')
→ "bXnXnX"

10. split(分隔符) 分割

按分隔符将字符串拆成数组，反向操作是 join

'a,b,c'.split(',')
→ ["a", "b", "c"]
'hello world'.split(' ')
→ ["hello", "world"]
'hello'.split('')
→ ["h", "e", "l", "l", "o"] (空字符串→逐字拆分)
// 第二个参数：限制返回数组长度
'a,b,c,d'.split(',', 2)
→ ["a", "b"]

11. concat(...) 拼接

拼接多个字符串，但更推荐用模板字符串

'Hello'.concat(' ', 'World', '!')
→ "Hello World!"
// 实际开发中更推荐模板字符串
const name = '张三', age = 25
`我叫${name}，今年${age}岁`
→ "我叫张三，今年25岁" (模板字符串更直观)

12. padStart / padEnd 填充

用指定字符补足长度，常用于格式化（时间、金额）

'5'.padStart(2, '0')
→ "05" (前面补0)
'123'.padStart(8, '*')
→ "*****123"
// 实战：格式化时间
const h = '9', m = '5', s = '3'
`${h.padStart(2,'0')}:${m.padStart(2,'0')}:${s.padStart(2,'0')}`
→ "09:05:03"

13. repeat(n) 重复

将字符串重复 n 次，n 为 0 返回空串

'Hi'.repeat(3)
→ "HiHiHi"
'*'.repeat(10)
→ "**********"
// 制作分隔线
'━'.repeat(40)
→ "━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━"

14. search / match / matchAll 正则查找

search 返回匹配位置，match 返回匹配结果，matchAll 返回迭代器

'hello 123 world'.search(/\d+/)
→ 6 (第一个数字出现的位置)
'a1 b2 c3'.match(/\d+/g)
→ ["1", "2", "3"]
// match 不带 g：返回详细信息（捕获组、index等）
'hello 2024'.match(/(\d+)/)
→ ["2024", "2024", index: 6, ...]

15. String.fromCharCode / fromCodePoint 静态方法

从编码值创建字符，反向操作是 charCodeAt / codePointAt

String.fromCharCode(65, 66, 67)
→ "ABC"
// 生成字母表
Array.from({length: 26}, (_, i) => String.fromCharCode(65 + i))
→ ["A","B","C",...,"Z"]

16. 综合实战 常见场景

// 1. 隐藏手机号中间4位
const phone = '13812345678'
phone.slice(0, 3) + '****' + phone.slice(-4)
→ "138****5678"
// 2. 首字母大写
const word = 'hello'
word[0].toUpperCase() + word.slice(1)
→ "Hello"
// 3. 驼峰命名转换 (hello-world → helloWorld)
function toCamel(str) {
  return str.split('-').map((w, i) => i === 0 ? w : w[0].toUpperCase() + w.slice(1)).join('')
}
toCamel('font-size-color')
→ "fontSizeColor"
// 4. 验证邮箱格式
function isEmail(str) {
  return str.includes('@') && str.includes('.') && !str.startsWith('@')
}
isEmail('test@qq.com')
→ true

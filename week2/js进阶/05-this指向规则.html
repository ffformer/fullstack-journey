05 — this 指向规则

JS 的 this 不是在定义时确定的，而是在函数被调用时动态绑定的。一共 5 种规则。

一、五种绑定规则速查

优先级规则this 指向示例
1（最高）new 绑定new 创建的实例对象new Foo() → this = 新对象
2显式绑定call/apply/bind 指定的对象fn.call(obj) → this = obj
3隐式绑定调用时 . 前面的对象obj.fn() → this = obj
4默认绑定undefined（严格模式）或 windowfn() → this = undefined
—（特殊）箭头函数定义时外层作用域的 this() => {} → this 固定不变

二、规则1：默认绑定

独立函数调用 — this = undefined（严格模式）/ window（非严格）

function sayHi() {
  console.log(this); // undefined（严格模式）或 window（非严格）
}
sayHi(); // ← 没有 . 没有 call 没有 new，走默认绑定

三、规则2：隐式绑定

通过对象调用 — this = . 前面的对象

const user = {
  name: '张三',
  greet() { console.log(this.name); }
};
user.greet(); // this = user → "张三"

// ⚠️ 坑：方法被赋值后丢失 this
const fn = user.greet;
fn(); // this = undefined → 报错！（this.name 读取失败）

四、规则3：显式绑定（call / apply / bind）

自己指定 this — call/apply 立即执行，bind 返回新函数

function greet(greeting, punctuation) {
  console.log(greeting + ', ' + this.name + punctuation);
}

const person = { name: '李四' };

greet.call(person, '你好', '!');   // this=person, 参数逐个传
greet.apply(person, ['嗨', '~']);  // this=person, 参数用数组传
const bound = greet.bind(person, 'Hello'); // 返回新函数，预设 this 和部分参数
bound('!!!'); // "Hello, 李四!!!"

五、规则4：new 绑定

new 运算符 — this = 新创建的实例对象

function Person(name) {
  // new 调用时，this 绑定到新创建的空对象
  this.name = name;
}
const p = new Person('王五');
console.log(p.name); // "王五" — this 指向 p 这个实例

new 做了 4 件事（Day 9 下午会手写）：

  1. 创建一个空对象 {}
  2. 把这个对象的 __proto__ 指向构造函数的 prototype
  3. 把构造函数的 this 绑定到这个对象
  4. 返回这个对象（除非构造函数自己 return 了引用类型）

六、箭头函数 — 没有自己的 this

箭头函数的 this 在定义时"捕获"外层作用域的 this，之后永不改变

const user = {
  name: '赵六',
  greet() {
    // 普通函数 — this 取决于调用方式
    setTimeout(function() {
      console.log('普通函数 this.name:', this.name); // undefined
    }, 10);

    // 箭头函数 — this 继承外层 greet() 中的 this
    setTimeout(() => {
      console.log('箭头函数 this.name:', this.name); // "赵六"
    }, 10);
  }
};
user.greet();

七、综合练习：判断 this 指向

以下代码中的 this 分别指向谁？

const obj = {
  name: 'obj',
  show() {
    console.log('1:', this);        // ?
    function inner() {
      console.log('2:', this);      // ?
    }
    inner();
    const arrow = () => console.log('3:', this); // ?
    arrow();
  }
};
obj.show();

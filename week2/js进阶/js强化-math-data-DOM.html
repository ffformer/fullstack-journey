一、Math 对象

Math 是内置对象，不是构造函数（不能用 new），所有属性和方法都是静态的

1. 数学常量

Math 提供了常用的数学常量

Math.PI
→ 3.141592653589793
Math.E
→ 2.718281828459045（自然对数的底）
Math.SQRT2
→ 1.4142135623730951（2 的平方根）

2. 取整与绝对值

abs 取绝对值，ceil 向上取整，floor 向下取整，round 四舍五入

Math.abs(-42)
→ 42
Math.ceil(3.1)
→ 4 // 向上取整
Math.floor(3.9)
→ 3 // 向下取整
Math.round(3.5)
→ 4 // 四舍五入，.5 时向正无穷

3. max / min — 最大最小值

接收多个参数，返回其中的最大值或最小值

Math.max(10, 25, 3, 88)
→ 88
Math.min(10, 25, 3, 88)
→ 3
// 实战：取数组最大/最小值（用展开运算符）
const arr = [5, 12, 8, 3]
Math.max(...arr)
→ 12

4. random — 随机数

返回 [0, 1) 之间的伪随机浮点数，0 ≤ x < 1

Math.random()
→ 0.734...（每次不同）
// 1~10 随机整数（含1和10）
Math.floor(Math.random() * 10) + 1
→ 1~10 之间的整数
// 通用公式：min~max 随机整数（含两端）
function randInt(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min
}

5. pow / sqrt — 幂与平方根

pow(底数, 指数) 求幂；sqrt 求平方根；cbrt 求立方根

Math.pow(2, 10)
→ 1024
Math.sqrt(64)
→ 8
Math.cbrt(27)
→ 3 // 立方根
// ES7 幂运算符 **
2 ** 10
→ 1024（等价于 Math.pow(2,10)）

6. 对数与指数函数

log = 自然对数(ln)；log10 = 以10为底；log2 = 以2为底

Math.log(Math.E)
→ 1（ln(e) = 1）
Math.log10(100)
→ 2
Math.exp(1)
→ 2.718...（e¹）

7. 符号判断与截断

sign 判断正负零；trunc 直接去掉小数部分

Math.sign(42)
→ 1（正数）
Math.sign(-42)
→ -1（负数）
Math.trunc(3.9) / Math.trunc(-3.9)
→ 3 / -3 // 与 floor 的区别：floor(-3.9) = -4

二、Date 对象

Date 是构造函数，需要用 new 创建实例来表示某个时刻

1. 创建 Date 实例

四种常见方式创建日期对象

// 当前时间
new Date()
→ 当前日期时间
// 时间戳（毫秒）
new Date(0)
→ 1970-01-01T00:00:00.000Z（Unix 纪元）
// 日期字符串
new Date('2024-05-20')
→ 2024年5月20日
// 年月日时分秒（月份从0开始！）
new Date(2024, 4, 20, 14, 30, 0)
→ 2024-05-20 14:30:00 // 月份 4 = 5月

2. get 系列 — 获取日期成分

注意 getMonth() 返回 0~11，getDay() 返回 0(周日)~6(周六)

const d = new Date('2024-05-20T14:30:00')
d.getFullYear()
→ 2024
d.getMonth()
→ 4 // ⚠️ 0=1月，4=5月！
d.getDate() / d.getDay()
→ 20 (号) / 1 (周一) // 0=周日
d.getHours() / d.getMinutes() / d.getSeconds()
→ 14 / 30 / 0

3. set 系列 — 设置日期成分

set 方法会直接修改原 Date 对象（类似数组的修改方法）

const d = new Date('2024-01-01')
d.setFullYear(2025)
d.setMonth(5) // 设为6月
→ d 变为 2025-06-01
// set 支持自动进位
const d2 = new Date('2024-01-31')
d2.setMonth(1) // 设为2月，但没有31天
→ 自动变为 2024-03-02

4. 时间戳（Timestamp）

从 1970-01-01 00:00:00 UTC 到现在的毫秒数，常用于计算时间差

Date.now()
→ 1716...（当前时间戳，13位毫秒）
new Date().getTime()
→ 等价于 Date.now()
// 计算代码执行时间
const start = Date.now()
// ... 执行一些代码 ...
console.log(`耗时: ${Date.now() - start}ms`)

5. 日期转字符串

不同方法输出不同格式的日期字符串

const d = new Date('2024-05-20T14:30:00')
d.toISOString()
→ "2024-05-20T06:30:00.000Z" // UTC 时区
d.toLocaleString()
→ "2024/5/20 14:30:00" // 本地格式
d.toLocaleDateString('zh-CN')
→ "2024/5/20" // 中文日期格式

6. 日期加减运算

计算两个日期之间的天数、加减天数等

// 两个日期相差多少天
const d1 = new Date('2024-05-01')
const d2 = new Date('2024-05-20')
const diff = (d2 - d1) / (1000 * 60 * 60 * 24)
→ 19（天） // 日期相减自动转时间戳
// 3天后是什么日期
const d = new Date()
d.setDate(d.getDate() + 3)
→ 三天后的日期 // setDate 自动进位处理跨月

三、DOM 概述

DOM（Document Object Model）是浏览器将 HTML 文档解析成的树形对象结构，JS 通过 DOM API 操作页面

1. 什么是 DOM？

HTML 是文本，DOM 是浏览器解析后的对象模型。可以把它理解为一棵"节点树"

// HTML:
// <body>
// <h1>标题</h1>
// <p>段落</p>
// </body>
→ body 是父节点，h1 和 p 是它的子节点，形成树形结构
// JS 中访问整个文档的入口
console.dir(document)
→ 展开可看到完整的 DOM 树

2. 选择 DOM 元素（核心）

querySelector 是现代最推荐的方式，支持 CSS 选择器语法

// 通过 ID 选择
document.getElementById('app')
→ 返回单个元素或 null
// 通过 CSS 选择器选第一个
document.querySelector('.card')
→ 第一个匹配的元素
// 通过 CSS 选择器选全部
document.querySelectorAll('.card')
→ NodeList（类数组，可用 forEach）
// 其他传统方法（了解即可）
document.getElementsByClassName('item')
document.getElementsByTagName('div')

3. 读写元素内容

textContent 只读文本（安全）；innerHTML 解析 HTML（有 XSS 风险）

// 读取/设置纯文本（推荐）
const el = document.querySelector('h1')
el.textContent = '新标题'
→ 页面上标题立即变为"新标题"
// 读取/设置 HTML（注意 XSS）
el.innerHTML = '<strong>加粗</strong>'
→ 解析为 HTML，显示加粗文字
// ⚠️ 用户输入绝不要直接插 innerHTML！
// 恶意脚本：<img src=x onerror="alert(1)">

4. 读写元素属性

标准属性直接用点语法；自定义属性用 getAttribute / setAttribute

// 标准属性直接读写
const img = document.querySelector('img')
img.src = 'new.jpg'
img.alt = '描述文字'
// 自定义属性（data-*）
// <div data-id="123"></div>
el.dataset.id // → "123"
// classList 操作类名
el.classList.add('active')
el.classList.remove('hidden')
el.classList.toggle('dark')
el.classList.contains('active') // → true/false

5. 操作内联样式

style 属性读写的是内联样式，驼峰命名对应 CSS 属性名

el.style.color = 'red'
el.style.backgroundColor = '#0f172a' // CSS: background-color
el.style.fontSize = '20px' // CSS: font-size
el.style.display = 'none' // 隐藏元素
// 获取计算后的样式（包含 CSS 文件中的样式）
const styles = getComputedStyle(el)
styles.color
→ 返回最终生效的颜色值

6. 创建与插入节点

createElement 创建空元素，然后通过 appendChild / append 插入 DOM 树

// 创建新元素
const div = document.createElement('div')
div.textContent = '我是新创建的'
document.body.append(div)
→ 页面出现新 div
// 插入到某个元素前面
parent.insertBefore(newEl, referenceEl)
// 删除元素
el.remove() // 直接从 DOM 中移除

7. 事件基础

事件是用户与页面交互的方式（点击、输入、滚动等），addEventListener 监听事件

// 绑定点击事件
btn.addEventListener('click', () => {
  console.log('按钮被点击了!')
})
// 常用事件类型
'click' — 鼠标点击
'input' — 输入框内容变化（实时）
'change' — 表单值改变并失去焦点
'submit' — 表单提交
'keydown' — 键盘按下
'DOMContentLoaded' — DOM 树构建完毕

8. 事件对象 event

事件回调的第一个参数是 event 对象，包含事件相关信息

el.addEventListener('click', (event) => {
  event.target // 触发事件的元素
  event.type // 事件类型，如 "click"
  event.preventDefault() // 阻止默认行为（如 a 标签跳转）
  event.stopPropagation() // 阻止事件冒泡
})

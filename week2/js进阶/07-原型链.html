07 — 原型链（Prototype Chain）

一、三个关键属性

属性属于谁指向什么通俗理解
prototype构造函数（函数才有）原型对象"我创建的实例长什么样"
__proto__所有对象构造函数的 prototype"我的模板是谁"
constructor原型对象构造函数本身"谁创建了我"

二、三角关系图

person 实例
    │
    │ .__proto__ ─────────┐
    │                          ▼
    │              Person.prototype（原型对象）
    │                          │
    │                          │ .constructor ──┐
    │                          │                  │
    │                          ▼                  ▼
    │              Person（构造函数）
    │                          │
    │                          │ .prototype ─────┘
    │                          │
    └──────────────────────────┘
      person.__proto__ === Person.prototype ✅

继续向上：
Person.prototype
    │
    │ .__proto__ ──────────▶ Object.prototype
    │                                       │
    │                                       │ .__proto__ ──▶ null（终点）
    │                                       │
    └───────────────────────────────────────┘

这个 __proto__ 一条线串起来的链路，就是原型链。JS 通过它实现属性的继承查找。

三、原型链让属性查找可以"向上走"

当读取 person.name 时，JS 引擎的查找顺序：

function Person(name) {
  this.name = name; // 实例自身属性
}
Person.prototype.species = '人类'; // 原型上的共享属性

const p = new Person('张三');

console.log(p.name);     // "张三" — 自己的属性，直接找到
console.log(p.species);  // "人类" — 自己没有 → 上溯到 Person.prototype 找到
console.log(p.toString());// [object Object] — 继续上溯到 Object.prototype 找到
console.log(p.xxx);      // undefined — 一直找到 null 都没有

四、原型继承：让一个"类"继承另一个

寄生组合继承（面试标准答案）

// 父类
function Animal(name) {
  this.name = name;
}
Animal.prototype.eat = function() {
  console.log(this.name + ' 在吃东西');
};

// 子类
function Dog(name, breed) {
  Animal.call(this, name); // 1. 继承父类的实例属性
  this.breed = breed;
}
// 2. 继承父类的原型方法 — Object.create 切断引用
Dog.prototype = Object.create(Animal.prototype);
// 3. 修正 constructor 指向
Dog.prototype.constructor = Dog;
// 4. 在子类原型上添加自己的方法
Dog.prototype.bark = function() {
  console.log(this.name + ' 汪汪叫');
};

const d = new Dog('旺财', '金毛');
d.eat();  // "旺财 在吃东西" — 从 Animal.prototype 继承
d.bark(); // "旺财 汪汪叫" — Dog 自己的方法

五、instanceof 原理

obj instanceof Constructor 做了什么？

顺着 obj.__proto__ 链一直向上找，看能不能找到 Constructor.prototype。

function myInstanceOf(obj, Constructor) {
  let proto = Object.getPrototypeOf(obj); // 等于 obj.__proto__

  while (proto) {
    if (proto === Constructor.prototype) return true;
    proto = Object.getPrototypeOf(proto); // 继续向上
  }
  return false;
}

const d = new Dog('旺财', '金毛');
myInstanceOf(d, Dog);     // true
myInstanceOf(d, Animal);  // true（原型链向上找到了！）
myInstanceOf(d, Object);  // true（再往上）
myInstanceOf(d, Array);   // false

Flex 弹性盒子模型

核心思想：给容器设置 display: flex，容器里的子元素会自动变成弹性项目，沿主轴排列。

记住两个概念：容器（父元素）= 设置 flex 的人 · 项目（子元素）= 被排列的人

1. flex-direction — 主轴方向

决定子元素沿哪个方向排列。

flex-direction:
1️⃣
2️⃣
3️⃣

2. justify-content — 主轴对齐

控制子元素在主轴方向上的分布方式。

justify-content:
A
B
C

3. align-items — 交叉轴对齐

控制子元素在交叉轴方向上怎么对齐（交叉轴 = 和主轴垂直的方向）。

切换主轴方向看看交叉轴的变化👇

align-items:
小字
大字
中字

4. flex-wrap — 是否换行

默认不换行（nowrap），子元素多了会被挤扁。设 wrap 后放不下就自动换行。

flex-wrap:
项目1
项目2
项目3
项目4
项目5
项目6

5. align-content — 多行时的交叉轴分布

只有 多行（flex-wrap: wrap）时才生效，控制行与行之间的间距。

align-content:
1
2
3
4
5
6
7
8

6. flex-grow — 子元素瓜分剩余空间

默认 0（不拉伸）。值越大，分到的剩余空间越多。

grow:0
grow:1
grow:2

7. align-self — 某个子元素单独做交叉轴对齐

可以打破 align-items 的规则，让某一个子元素单独对齐。

默认居中
align-self: flex-start
align-self: flex-end
align-self: stretch

8. order — 调整子元素显示顺序

默认都是 0，值越小越靠前。可以用纯 CSS 改顺序，不用动 HTML。

order:3 → 排第3
order:1 → 排第1
order:2 → 排第2

速查表

属性作用对象一句话解释
display: flex容器开启弹性布局
flex-direction容器主轴方向（row / column）
justify-content容器主轴上的对齐/分布
align-items容器交叉轴上的对齐（单行）
flex-wrap容器是否换行
align-content容器多行时行与行的分布
gap容器子元素之间的间距
flex-grow项目瓜分剩余空间的比例
flex-shrink项目空间不足时缩小的比例
align-self项目单个项目的交叉轴对齐
order项目排序（值越小越靠前）

提示：打开浏览器开发者工具（F12），在 Elements 面板选中 .flex-container，可以直接看到 flex 的可视化标注。

Flex 布局完全指南

Flex 布局是 CSS3 中最重要的布局方式之一。通过 display: flex 可以轻松实现一维布局，配合 justify-content 和 align-items 控制主轴和交叉轴的对齐方式。

CSS 盒模型深入理解

每个 HTML 元素都可以看作一个盒子，由 content → padding → border → margin 四层组成。box-sizing: border-box 是最实用的设置，让宽高包含 padding 和 border。

响应式设计入门

使用 @media 媒体查询，可以根据屏幕宽度切换布局。结合 Flex 的 flex-direction: column，可以让桌面端的多栏布局在移动端自动变成单栏。

### 本分类核心概念

- CSS 选择器优先级：ID > 类 > 元素
- display 三态：block（独占一行） / inline（不接受宽高） / inline-block（同行且接受宽高）
- 盒模型：content → padding → border → margin，`box-sizing: border-box` 让宽高包含 padding 和 border
- 浮动：元素脱离正常流，父元素会高度塌陷，用 `overflow: hidden` 清除
- 定位：relative 相对自己原位偏移，absolute 相对最近的已定位祖先，fixed 相对视口
- Flex 布局：`display: flex` + `flex: N` 按比例分配空间
- 响应式：`@media (max-width: 768px)` 切换布局
